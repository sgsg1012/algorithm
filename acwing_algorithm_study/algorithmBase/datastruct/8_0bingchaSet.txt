并查集
    8_1 合并集合
    8_2 连通块中点的数量
    8_3 食物链

操作：
    将两个集合合并
    询问两个元素是否在同一个集合中

基本原理：
    用一棵树来表示一个集合，树根的编号就是集合的编号
    每个节点存储它的父节点

如何判断树根：
    p[x]==x
如何求x的集合编号：
    while(p[x]!=x) x=p[x]; //可能会耗时，可优化  路径压缩
    优化后代码：
        int find(x)
        {
            if(x!=p[x]) x=find(p[x]);
            return p[x];
        }
    // 太妙了！！！！
如何合并两个集合：
    把树1的根节点插入树2某个节点下


有时并查集要维护一些属性来满足需求
